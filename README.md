# Company Task Management System

A web-based task management system designed for companies to organize and track employee tasks.
The platform supports role-based access for admins and employees.

## Features
- Admin task assignment
- Tasks visible only to assigned employees
- Task status updates (In Progress / Completed)
- Employees can create personal tasks
- Separate dashboards for admin and employees

## Roles
- Admin (Manager): Assign and manage employee tasks
- Employee: View, update, and create tasks

## Tech Stack
- PHP
- MySQL
- HTML / CSS
- JavaScript
